# 医院管理系统 java269

## 项目概述

医院管理系统是基于SpringBoot和Vue的前后端分离架构，适用于医疗机构的信息化解决方案。系统提供了患者、医生和管理员三种角色，满足医院日常运营的需求。

## 技术栈

- **架构**: B/S、MVC
- **开发环境**: 
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- **技术栈**: 
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 用户模块
- 用户登录

### 患者模块
- 预约挂号
- 我的挂号
- 报告单导出
- 住院信息
- 患者个人信息

### 医生模块
- 病患诊断
- 历史挂号列表
- 住院申请管理
- 医生个人信息

### 管理员模块
- 医生信息管理
- 患者信息管理
- 挂号信息管理
- 药物信息管理
- 检查项目管理
- 病床信息管理
- 排班信息管理
- 数据统计分析
